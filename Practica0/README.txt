En esta carpeta se encuentran los archivos asociados a la práctica 0 del
curso de EDD.

El archivo generaPractica.sh crea una carpeta con los subdirectorios DOCS, BIN,
SRC y el archivo de texto plano README.txt, recibe como parámetro el nombre de
la carpeta a crear.

El archivo generarEnvio mueve los archivos .class del directorio /SRC a /BIN,
genera la documentación java de los .java en /SRC y comprime la carpeta
principal, también recibe el nombre de la carpeta que contiene la práctica.

Para ejecutar los scripts es necesario dar permiso con chmod +x nombre, y
ejecutar.
